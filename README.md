# Pritech Task Manager

A task management app built with React Native + Expo. Runs in the browser — no device or simulator needed.

---

## Requirements

- [Node.js 18+](https://nodejs.org)
- [Git](https://git-scm.com)

---

## Run locally

```bash
# 1. Clone
git clone https://github.com/rinormaliqi/PRITECH.git
cd pritech

# 2. Install
npm install --legacy-peer-deps

# 3. Start
npm run web
```

A browser tab will open automatically at `http://localhost:8081`.

---

## That's it.

No accounts, no backend, no environment variables. Everything runs locally in the browser.
